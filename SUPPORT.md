# Support

Cadence Lite is free, self-hosted software.

I’m sharing it because I want people to have access to a more transparent, user-owned AI companion setup without needing to buy Cadence Core if they can't justify the cost. That said, Lite is not a paid support product, and I can’t realistically offer guaranteed one-to-one setup help or ongoing troubleshooting for every install.

I’ll help where I can, especially with clear bugs or issues that may affect other users, but support for Cadence Lite is offered on a best-effort basis.

## Bug Reports

If you find a reproducible bug, please use GitHub Issues.

When reporting a bug, it helps to include:

- what you were trying to do
- what happened instead
- any relevant error message or log output
- your rough setup context, if it matters

Please do **not** post private personal memory content, bot tokens, API keys, license keys, API responses, or any other sensitive data in public bug reports.

## Setup Help

This repo is primarily set up for deploymet and bug reporting, not general one-to-one setup support.

For setup walkthroughs and guidance, please use the available documentation and any additional resources published through official Cadence channels, such as Patreon and Discord.

If you need more direct setup help, Cadence Core includes a higher level of guidance and support because it is the paid version of the system. Even there, support is not 24/7 or unlimited, but paid users will naturally be prioritised where hands-on help is needed.

## Support Expectations

There is no guaranteed support response time for Cadence Lite.

I’ll do my best to keep the project maintained, fix reproducible bugs, and clarify documentation where needed, but please understand that free software does not come with guaranteed personal support.

## Contributions

This repo is not currently open for outside code contributions.

Feedback and bug reports are welcome. Pull requests are not being actively invited at this stage.
