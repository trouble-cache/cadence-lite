# Support

Cadence Lite is currently beta software.

## Bug Reports

If you find a reproducible bug, please use GitHub Issues.

When reporting a bug, it helps to include:

- what you were trying to do
- what happened instead
- any relevant error message or log output
- the rough setup context, if it matters

Please do **not** post private personal memory content, bot tokens, API keys, or other sensitive data in public bug reports.

## Setup Help

This repo is primarily set up for bug reporting, not general setup support.

For setup walkthroughs and guides, please use the available documentation and any additional guidance published through Cadence channels such as Patreon.

## Support Expectations

There is currently no guaranteed support response time.

I’ll do my best to keep things moving, but Cadence Lite is still in beta and support is offered on a best-effort basis.

## Contributions

This repo is not currently open for outside code contributions.

Feedback and bug reports are welcome. Pull requests are not being actively invited at this stage.
